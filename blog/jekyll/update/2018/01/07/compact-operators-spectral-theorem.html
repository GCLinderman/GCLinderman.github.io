Spectral Theorem for Compact Operators

Jan 7, 2018

Compactness

In a metric space, it is equivalent for a set to be compact and sequentially compact. Let $X$ be a metric space equipped with a distance function $d (\cdot, \cdot )$.

Compact

A set is compact if every cover has a finite subcover. If $ B $ is compact, then it is closed and bounded. Assuming $ B $ is compact, we first prove its complement is open. Consider a point $ p \in B^c $, and define an open cover $U_n = \{ q : d(p,q) > \frac{1}{n} \}$. Note that these sets are nested: $U_1 \subset U_2 \subset … $, and because $\{ U_n \}$ covers $B$, there is a finite subcover. That is, there exists some $m$ such that $B \subset U_m$. Therefore, $N_{(1/m)}(p) \in B^c$, proving that $B^c$ is open, which gives that $B$ is closed.

Now we prove that $B$ is bounded. This can be seen by letting $\{V_n \}$ be a cover for $B$ defined by $V_n = N_n(p)$ for any point $p \in B$. This cover is also nested, and it has a finite subcover, so there exists some $m$ for which $B \subset N_m(p)$, meaning that $B$ is bounded.

So, when do we get the other direction? In $\mathbb{R}^n$, we have it in Heine-Borel: A subset of $\mathbb{R^n}$ or $\mathbb{C^n}$ that is closed and bounded is compact.

What is it about Euclidean space that gives us this very nice result? First, is completeness. For example, let our space be $\mathbb{Q}$ with the usual metric. $\mathbb{Q} \cap [0,1]$ is bounded and closed in $\mathbb{Q}$, but not compact.

Second, is that it is finite. In a finite dimensional space, every bounded set is totally bounded, meaning that it can be covered by finitely many subsets of every fixed size. In an infinite dimensional space, that is not necessarily true.

Well, we can generalize it to arbitrary metric spaces by strengthening boundedness to total boundedness and closedness to completeness.

A subset of $X$ that is complete and totally bounded is compact.

In some very useful metric spaces, however, there are closed and bounded sets which are not compact (e.g. for a Hilbert space example, see here).

Sequentially Compact

A set is sequentially compact if every sequence has a convergent subsequence. As mentioned above, A subset of a metric space is compact if and only if it is sequentially compact.

Bolzano-Weierstrass Any closed or bounded set of $\mathbb{R}^n$ or $\mathbb{C}^n$ is sequentially compact.

Because of the equivalence between compactness and sequential compactness in a metric space, Heine-Borel and Bolzano-Weierstrass are equivalent.

Compact Operators

We begin by recalling the definition of a compact operator

A linear transformation $T : \mathscr{K} \rightarrow \mathscr{K}$ is compact if $T ( \text{ball } \mathscr{H} )$ has compact closure in $\mathscr{K}$.
